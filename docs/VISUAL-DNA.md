# Dr. Fadime Cenik - Visual DNA

## Logo Analysis

### Color Extraction
- **Deep Medical Blue:** `#1e3a5f` - Primary brand color (figure, name, subtitle)
- **Warm Gold/Bronze:** `#c9a962` - Accent divider line, warmth
- **Ivory/Off-White:** `#faf8f5` - Background, breathing space

### Typography
- **Name:** Serif font (Playfair Display style) - Authority, trust
- **Subtitle:** Sans-serif with generous letter-spacing - Modern, clean
- **Characteristics:** Calm, premium, timeless

### Human Figure Symbol
- Abstract human in motion (rehabilitation/recovery)
- Connected joints (circles) representing body points
- Curved, organic lines suggesting movement
- Symbolizes: healing, recovery, human care

## Design System

### Color Palette
```css
--color-deep-blue: #1e3a5f;      /* Primary - headings, structure */
--color-deep-blue-light: #2a4a6f; /* Hover states */
--color-deep-blue-dark: #152d4a;  /* Darker variant */
--color-gold: #c9a962;            /* Accent - highlights, dividers */
--color-gold-light: #d4b872;      /* Hover accent */
--color-gold-dim: rgba(201, 169, 98, 0.2); /* Subtle backgrounds */
--color-ivory: #faf8f5;           /* Background */
--color-ivory-warm: #f5f2ed;      /* Secondary background */
--color-text: #1e3a5f;            /* Primary text */
--color-text-muted: #5a6f85;      /* Secondary text */
```

### Typography System
```css
--font-heading: 'Playfair Display', Georgia, serif;
--font-body: 'Inter', -apple-system, sans-serif;
--letter-spacing-wide: 0.15em;    /* For subtitles */
--letter-spacing-normal: 0.02em;  /* For body */
```

### Motion Principles
- Slow, intentional animations (0.6s - 1.2s)
- Organic easing curves
- Subtle transforms (no aggressive movements)
- Breathing rhythm (4-6s cycles)

### Shape Language
- Rounded corners (organic, human)
- Soft shadows (no harsh edges)
- Curved dividers (matching logo figure)
- Circle accents (joint points from logo)

## UX Philosophy

### Entry Experience
1. Soft ivory background
2. Subtle logo presence (not dominant)
3. Centered greeting with typing effect
4. 3 intent choice cards with blue structure + gold accents

### Navigation
- No traditional navbar
- AI Guide Orb (bottom right)
- Intent-based content flow
- Modules instead of pages

### Emotional Design
- Medical: Professional, trustworthy
- Human: Warm, caring, personal
- Intelligent: Adaptive, responsive
- Calm: No urgency, no pressure

---

## Implementation Rules

### DO
✅ Use Deep Blue for all structural elements
✅ Use Gold sparingly for emphasis
✅ Maintain generous whitespace
✅ Keep animations slow and organic
✅ Use serif for authority (headings)
✅ Use sans-serif for clarity (body)

### DON'T
❌ Use loud or saturated colors
❌ Use sharp gradients
❌ Use aggressive animations
❌ Use trendy design patterns
❌ Overcrowd with information
❌ Make the logo dominant
